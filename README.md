## Вывод
В 1С:ERP успешно реализован полный цикл планирования: создан сценарий «Будянская сценарий» с месячной периодичностью, настроена структура планов (продаж, производства, закупок), заполнены и согласованы планы по товарам ноутбукам, мониторам и клавиатурам. Все документы утверждены, что подтверждает завершение этапа планирования. По результатам сформирован отчёт «Сбалансированность планов», показывающий, что обеспечение превышает потребность это перестраховка, но планы сбалансированы и готовы к исполнению. Работа полностью соответствует заданию и демонстрирует корректное управление цепочкой «продажи - производство - закупки».
